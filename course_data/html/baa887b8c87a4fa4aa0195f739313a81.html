Activité : Stocker, traiter et visualiser des données sur emoncms.Fr

Niveau de difficulté : ☆ (facile)

Ça y est, nos premières données ont été captées, formatées puis envoyées sur les Internets pour arriver sur notre serveur, il est temps de passer à la suite !

Dans le tutoriel suivant, nous allons utiliser emoncms pour stocker nos sources dans des flux. Nous verrons qu'il est possible d'appliquer des pré-traitements avant de stocker les données et enfin nous verrons comment visualiser nos flux. Let's go 👇

  1. Rendez-vous à l'adresse suivante : https://emoncms.fr/input/view (vous devez avoir créer un compte et être identifié), vous devriez voir une liste des Sources que vous avez créés. Cliquez sur l’icône de configuration 🔧 d'une des sources :

  2. Nous allons maintenant ajouter un traitement à notre source permettant de stocker toutes les valeurs reçues dans un flux. Ce traitement s'appelle Log to feed et nous allons choisir de l'appeler light_level et de sauvegarder une valeur toutes les 10 secondes :

  3. Cliquez sur le bouton « Ajouter » puis sur le bouton « Modifié, presser pour sauver » :


    Veuillez noter au passage qu'il existe de nombreux traitements possibles sur les données reçues et que ces traitements seront exécutées séquentiellement (les uns à la suite des autres). Pour citer quelques traitements qui pourraient vous être utiles :

    • Calibrage/conversion : il sera possible de multiplier ou d'additionner la valeur reçue avec un nombre donné par l'utilisateur.
    • Conversion énergie : si vous envoyez une puissance (exprimée en Watts), ce traitement permet de le stocker en énergie (exprimée en Watts par heure).
    • Autres traitements : Max daily permet de récupérer le maximum d'un flux sur la journée tandis que Min daily viendra stocker le minimum...

  4. Vous pouvez maintenant cliquer sur « Flux » dans le menu latéral et vérifier que votre flux a bien été créé :


    Dès lors, toutes les données envoyées seront horodatées et stockées dans ce flux !

  5. Maintenant, un peu de visualisation ! Cliquez sur « Graphiques » dans le menu latéral puis cochez la case correspondante à votre flux (dans mon cas : light_level) :

  6. Bravo, vous venez d'arriver à la fin de ce tutoriel ! Nous vous laissons parcourir par vous-même les différentes options de visualisation offertes par emoncms. Avant de passer à la section suivante, veuillez coller dans l'autocorrecteur ci-dessous votre clé d'API « Lecture uniquement » ⬇️

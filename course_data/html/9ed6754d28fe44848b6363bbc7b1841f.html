Activité : Le prochain métro

L'activité ci-dessous est notée et dispose de plusieurs versions avec des difficultés croissantes :

    ⭐️⭐️ : Facile
    ⭐️⭐️⭐️ : Moyen
    ⭐️⭐️⭐️⭐️ : Difficile

Pour obtenir la note maximale, vous devez valider les trois versions !

Contexte

Encore un exercice sur les rails, mais les rails de métro cette fois ! Nous voici à Rennes, ville de makers, pour ce TP : Baptiste travaille à IMT Atlantique à l'est de la ville et souhaite être notifié lorsque le métro B arrive à proximité de l'arrêt "Beaulieu - Université" portant le numéro 5052. Il est suffisamment proche pour pouvoir être à l’arrêt en quelques minutes après avoir claqué la porte de son bureau. Mais manque de bol, il a l’impression que le métro lui passe toujours sous le nez ! Pour ne plus jamais le rater, il aimerait mettre au point un petit dispositif qui le prévient quand son métro arrive !

Consignes

À partir des concepts abordés dans ce MOOC et du portail web qui met à disposition les informations temps-réels du métro rennais. Vous devrez :

  • Trouver l'url permettant de récupérer les deux prochains passages du métro ligne B à l'arrêt 5052 depuis le portail web du STAR.
  • Ensuite, écrivez un programme en utilisant l'interface FranceIOI ci-dessous qui devra :
    • Connecter la Galaxia à un réseau wifi en mode station.
    • Récupérer les horaires de passage de deux prochaines rames de métro ligne B à l'arrêt 5052 grâce à une requête HTTP GET sur l'url que vous avez trouvé plus haut.
    • Afficher ces horaires de passages grâce aux éléments JSON "arriveefirsttrain" et "arriveesecondtrain".
      ⚠️ Attention : les horaires sont précédés de la date que vous devrez enlever de la chaîne de caractères (veuillez vous référer à la documentation officielle python pour "trancher" un texte).
    • Dans la version ⭐️⭐️⭐️⭐️, vous devrez également allumer la LED RGB en rouge (R=255, G=0, B=0) quand le temps d’attente pour le prochain métro est inférieur à 2 minutes. Vous devrez utiliser l'élément "heureextraction" pour calculer ce temps d'attente !

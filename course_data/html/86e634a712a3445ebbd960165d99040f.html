Avant de se lancer dans ce cours, on s'est dit que quelques rappels issus du cours "Programmer un objet avec MicroPython" serait de bon ton ! En effet, les quatre semaines de ce MOOC ne sont pas destinées aux néophytes et, pour suivre ce cours, vous devez maîtriser les bases de la programmation Python : les variables, les conditions, les boucles, les fonctions et les listes ainsi que les modules spécifiques à MicroPython.

Du coup, si vous venez tout juste de terminer notre premier cours "Programmer un objet avec MicroPython" ou que vous programmez en Python tous les jours vous pouvez aller directement aller résoudre la première activité. Sinon, nous vous invitons à aller jeter un coup d’œil à la documentation du cours qui contient tout ce que dont vous aurez besoin 👇

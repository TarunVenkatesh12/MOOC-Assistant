Pour l'expérience, j'ai mesuré la consommation de la carte Galaxia lors de l’exécution du code ci-dessus afin de vérifier que la consommation diminuait lorsque l'ESP passait en sommeil profond. Les mesures ont été faites avec un détecteur de tension/intensité USB de ce type :

La fonction de ce détecteur est de pouvoir connaître en temps réel la tension et le courant consommés sur un port USB. La précision de ce détecteur est plutôt limitée mais permet de voir une très nette différence entre le mode sommeil profond et le mode normal :

Mode normalMode sommeil profond

On note donc une consommation au moins 8 fois inférieure lorsque le passe en mode sommeil profond. Vous trouverez dans le tableau ci-dessous quelques mesures données à titre indicatives afin de saisir les économies d'énergie possible :

CodeTension moyenneIntensité moyenne
Sommeil profond 5,20V 0,01A
Allumer une LED (écran allumé) 5,20V 0,05A
Envoyer une donnée (écran allumé) 5,20V 0,10A

On notera donc que notre champion de la consommation est le WiFi et qu'il faudra limiter un maximum son utilisation si nous alimentons notre module ESP sur batterie.

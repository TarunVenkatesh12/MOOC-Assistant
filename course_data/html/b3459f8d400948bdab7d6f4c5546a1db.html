Une fois que vous avez vérifié que votre code correspond à la consigne, voici les étapes à suivre pour valider votre code :

  1. Depuis l'interface Wokwi, copiez le code au format texte en utilisant la combinaison de touche Ctrl+C (sous Windows ou Linux) ou ⌘+C (sous Mac OS)
  2. Collez ce code en utilisant la combinaison de touche Ctrl+V (sous Windows ou Linux) ou ⌘+V (sous Mac OS) dans le champs texte de l'autocorrecteur.
  3. Cliquez enfin sur Valider !

C'est à vous 👇

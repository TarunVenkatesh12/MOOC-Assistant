Installer manuellement le module requests sur la Galaxia

Si vous disposez d'une vraie carte Galaxia, vous verrez que nos interfaces de développement viennent installer automatiquement le module additionnel requests sur votre carte programmable ! Mais sachez que vous pouvez installer ce module "à la main" et pour cela, vous pouvez suivre les instructions ci-dessous :

  1. Téléchargez le fichier requests.py et enregistrez ce fichier sur votre disque dur (ce fichier est issu du github de micropython).
  2. Branchez votre carte Galaxia sur votre ordinateur en utilisant le câble USB fourni.
  3. Votre carte Galaxia est détectée comme une clé USB pouvant stocker des fichiers. Faites glisser le fichier requests.py téléchargé à l'étape 1 depuis votre disque dur vers la racine de la carte Galaxia :

  4. Profitez maintenant de toutes les fonctions de ce module, à vous de jouer !

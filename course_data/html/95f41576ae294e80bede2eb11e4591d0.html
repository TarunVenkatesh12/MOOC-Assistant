Activité : Créer un compte sur emoncms.fr

Niveau de difficulté : ☆ (facile)

Voici un tutoriel qui détaille les étapes pour créer un compte et récupérer une clé d'API sur le site emoncms.fr, c'est parti ⬇️ :

  1. Rendez-vous à l'adresse suivante https://emoncms.fr/ et cliquez sur « s'inscrire »

  2. Remplissez le formulaire d'inscription avec vos informations (votre adresse email et votre nom d'utilisateur n'ont pas besoin d'être les mêmes que sur fun-mooc.fr) :

  3. Cliquez enfin sur le bouton « S'inscrire » puis, une fois authentifié, cliquez sur « Sources » dans la barre de menu latérale :

  4. Comme vous pouvez le voir, aucune donnée n'a été reçue : c'est normal vous venez juste de créer votre compte. Cliquez sur « Aide de l'API Sources » en haut à droite de la page :

  5. Vous pouvez voir en haut de cette page les deux clés d'API dont vous aurez besoin pour échanger des données entre votre objet connecté et le serveur emoncms. La première clé d'API appelée « Lecture uniquement » vous permettra de recevoir les données d'une source ou d'un flux mais vous devrez utiliser la clé d'API « Lecture & Écriture » pour envoyer des données :

  6. Nous allons ensuite créer une source où notre navigateur va jouer le rôle de client. Pour cela,  en utilisant les liens présents en dessous des clés d'API, dans la section « Publication des données dans EmonCMS ». Cliquez sur le premier lien et notez au passage les différents composants de l'url :


    En ouvrant ce lien, vous venez de créer trois nouvelles sources appelées power1 (ayant pour valeur 100), power2 (ayant pour valeur 200) et power3 (ayant pour valeur 300) raccrochées au nœud appelé emontx. En vous rendant à nouveau dans Sources, vous pouvez voir que les sources ont bien été créées :

  7. Bravo, vous venez d'arriver à la fin de ce tutoriel ! Pour vérifier que tout fonctionne, veuillez coller dans l'autocorrecteur ci-dessous votre clé d'API « Lecture uniquement » ⬇️

Merci d'avoir regardé cette vidéo ! Il est temps de mettre un peu les mains dans le code mais avant de se lancer, il va nous falloir réfléchir à ce qu'on veut coder ! En effet, dans la vidéo, Baptiste nous explique que pour que deux éléments communiquent, il faut qu'ils parlent la même langue ou plutôt les mêmes protocoles !

Et oui, impossible de communiquer si on ne parle pas la même langue. Nous chercherons donc à décrire dans les prochaines sections comment faire communiquer le module ESP de notre Galaxia avec un serveur web qui parle en HTTP qui est le protocole applicatif utilisé lorsque vous consultez une page web ! Or, pour parler le HTTP, il faut regarder les protocoles qui se trouvent en dessous. Et oui, comme Baptiste le dit dans la vidéo : pour qu'un protocole assure sa fonction, il faut que la couche du dessous marche aussi ! On commencera donc par connecter les couches les plus basses pour ensuite remonter dans les couches !

Première étape : le WiFi

On commence donc par s'intéresser à la couche la plus basse et dans notre cas, il s'agit du protocole WiFi. En pratique, pour connecter la Galaxia à un réseau wifi, il faut d’abord s’assurer que le point d’accès wifi auquel on souhaite se connecter est à portée et vu par le module ESP de la Galaxia.
